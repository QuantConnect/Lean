# Templates Directory

> **Reusable templates for creating new strategies.**

---

## Available Templates

| File | Purpose |
|------|---------|
| `STRATEGY.template.md` | Strategy specification template |
| `config.template.json` | Configuration file template |

---

## Usage

To create a new strategy:

```bash
# Copy the entire template folder
cp -r _templates/strategy-template ../active/my-new-strategy/

# Or copy individual files
cp STRATEGY.template.md ../active/my-strategy/STRATEGY.md
cp config.template.json ../active/my-strategy/config.json
```
